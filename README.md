# Appian Ops Forecast Simulator

## Overview
A predictive operations dashboard that forecasts backlog growth,
bottlenecks, and SLA breach risk based on live workload and workforce inputs.

## Features
- Backlog growth prediction
- SLA breach risk classification
- Capacity vs demand simulation

## Tech Stack
- Python (Streamlit)
- HTML / CSS / JavaScript

## How to Run
```bash
pip install streamlit
streamlit run app.py
